# Políticas de Privacidad

Este documento describe cómo la aplicación CashFlow App maneja los datos del usuario.

## Datos recopilados
- La aplicación solo almacena localmente la información financiera que el usuario introduce.
- No se envían datos personales o financieros a servidores externos.

## Uso de los datos
- Los datos guardados se utilizan exclusivamente para mostrar balances, presupuestos y reportes dentro de la app.
- No se comparten los datos con terceros.

## Retención y eliminación
- El usuario puede limpiar o restablecer los datos de la aplicación en cualquier momento eliminando el almacenamiento local del navegador.

## Contacto
Si tienes preguntas sobre estas políticas, abre un issue en este repositorio.
