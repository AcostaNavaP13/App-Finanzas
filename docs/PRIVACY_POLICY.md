# Políticas de Privacidad

_Última actualización: 2024-08-22_

Este repositorio contiene la documentación sobre cómo CashFlow App maneja la información financiera que ingresas. Estas políticas se alojan solo como nota informativa en GitHub para que puedas consultarlas antes de usar la aplicación.

## Alcance
- Aplica a los datos que se almacenan en tu navegador al usar CashFlow App.
- No cubre servicios externos ni integraciones de terceros (no se usan actualmente).

## Datos que se guardan
- Los datos financieros y categorías que introduces en la app (p. ej., ingresos, gastos, presupuestos).
- Preferencias locales como filtros o vistas seleccionadas.
- **No** se recopilan datos personales identificables ni se envían datos a servidores externos.

## Cómo se usan los datos
- Se emplean únicamente para calcular balances, presupuestos y reportes dentro de tu navegador.
- No se comparten con terceros ni se utilizan con fines de analítica o publicidad.

## Dónde se almacenan
- Toda la información se guarda en el almacenamiento local del navegador del dispositivo donde usas la app.
- Si usas la app en varios dispositivos o navegadores, los datos no se sincronizan entre ellos.

## Retención y eliminación
- Los datos permanecen en tu navegador hasta que los elimines.
- Puedes borrar la información desde las opciones de borrar datos/cache de tu navegador o usando cualquier función de restablecimiento que incluya la app.

## Permisos y cookies
- La app no solicita permisos especiales ni utiliza cookies de terceros.
- Puede usar almacenamiento local (LocalStorage/IndexedDB) para guardar tus datos y preferencias.

## Cambios a estas políticas
- Las actualizaciones se publicarán en este mismo documento. Revisa la fecha de actualización para ver la versión vigente.

## Contacto
Si tienes preguntas o sugerencias sobre estas políticas, abre un issue en este repositorio.
